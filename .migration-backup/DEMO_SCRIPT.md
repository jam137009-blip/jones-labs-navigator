# Public Demo Script — Under Three Minutes

## 0:00–0:20 — Problem

“Most AI assumes you already know what to ask. But people often possess an intention before they possess the words, prompts, or expertise needed to express it.”

## 0:20–0:45 — Imperfect intention

Enter: “I have an idea that could help people make better decisions, but it keeps growing and I don’t know what part to build first.”

Explain: “No prompt engineering. No expert selection. Just the intention.”

## 0:45–1:05 — Pattern recognition

Show the three concise hypotheses. Select the closest direction and explain that each option must represent a materially different interpretation. Briefly show the correction path.

## 1:05–1:25 — Precision question

“Navigator asks only if one question can materially reduce the greatest remaining uncertainty.” Answer the single generated question.

## 1:25–2:15 — Signature insight

Pause on **The Question Beneath the Question**. Then show current reality, intended outcome, primary constraint, visibly labeled hypotheses, no more than three possible directions, and the one evidence-producing next move.

## 2:15–2:40 — Authority and evidence

Show confidence, what could change the interpretation, and the challenge test. Continue to the single evidence-producing commitment.

## 2:40–3:00 — Close

“Navigator asks only what can change the recommendation, stops when understanding is sufficient, and gives the user one evidence-producing next move. The goal is not to make AI look capable. It is to make the user more capable.”
